Fleet Driver Management

This module helps monitoring fleet driver attendance.

Overview

This module is exclusively designed for monitoring the fleet driver and managing their attendance and daily payments(Extension of https://apps.odoo.com/apps/modules/14.0/fleet_rental/) .
